This program is written in java and compiled/run using eclipse.
The program can be run from the command like using the following commands

javac kMeans.java
java kMeans

The program reads the input file provided, clusters the data and then 
writes back to an output file the data as follows
x-value|y-value|ClusterNumber

It calculates the numbers of clusters (k) provided in each file.
